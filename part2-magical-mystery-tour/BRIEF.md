# Magical Mystery Setlist

Build a tool for planning a live concert setlist.

A band needs to arrange their songs into a set with a target duration. They
should be able to pick songs from a catalog, reorder them, and see at a
glance whether the set is too long, too short, or just right. Think of it
as a to-do list, but for a concert.

A starter song catalog is provided in `data/songs.csv` with titles, artists,
durations, genres, tempos, and energy levels. Import it into the database on
first run. Users should also be able to add their own songs to the catalog.
